# Financial Market Data Analysis

## Objective
To analyze and model financial datasets to extract actionable market insights, track performance, and validate data accuracy. This project was developed during an advanced Financial Data Analysis program at Tsinghua University and demonstrates the ability to manage nuanced financial systems and identify data-driven trends.

## Tech Stack
* **Language:** Python
* **Data Manipulation:** Pandas, NumPy
* **Visualization:** Matplotlib, Seaborn
* **Environment:** Jupyter Notebook

## Key Results
* Processed and cleaned complex financial datasets to identify underlying market trends and anomalies.
* Designed structured analytical models to evaluate financial metrics.
* Translated raw financial data into clear, executive-ready visualizations to support technical and non-technical decision-making.
